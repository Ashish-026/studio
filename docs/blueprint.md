# **App Name**: Mandi Monitor

## Core Features:

- User Authentication: Secure login with username and password for different user roles (admin, register users).
- Role-Based Access: Restrict access to features and data based on user roles (e.g., only admins can enter target allocation data).
- Register Selection: UI to choose from OSCSC, Private, and Labour registers.
- Target Allocation (Admin Only): Admin panel to enter mandi name, date-wise target allocation. Calculates and displays the total target allocation at the top.
- Paddy Lift Entry: Form to input mandi name, farmer name, total paddy received, and mandi weight.
- Data Summarization: Aggregates data to display: target allotted, physical paddy lifted, balance to be lifted, and non-physically lifted.

## Style Guidelines:

- Primary color: Dark slate blue (#37474F) for a professional and trustworthy feel.
- Background color: Light gray (#ECEFF1), a desaturated version of the primary color, to provide a clean backdrop.
- Accent color: Deep orange (#FF7043), an analogous color for highlighting important actions.
- Font pairing: 'Inter' for body text (sans-serif) to provide a clean and modern feel and 'Playfair' for Headings (serif).
- Use simple, outline-style icons for navigation and actions. Ensure icons are consistent and intuitive.
- Maintain a clean and organized layout with clear separation between sections. Use cards or similar containers to group related information.
- Use subtle animations for loading states and transitions to improve user experience. Avoid distracting or unnecessary animations.
